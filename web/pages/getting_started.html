First steps

  • Clone or download explanes.
  • Read the documentation.
  • Install optional requested external softwares if needed.
  • Experiment with the demonstrations.
  • Build your first experiment !!.

Requisites

expLanes operates in the Matlab environment. A recent version of this software (> R2014) is needed as well as the Statistical toolbox. The Parallel toolbox is needed for in-session multi-core processing. As much as possible, expLanes uses built-in tools to manipulate data. However some functions are handled through the use of basic Unix tools:

  • pdflatex: handles compilation of reports written in LATEX.
  • rsync: allows backup, generation of bundles and data communication between hosts.
  • ssh: allows secure connection between hosts.
  • screen: allows handling of several detached terminals for background processing.

If you wish to deploy your experiment across multiple hosts, those tools must be available on each host. However, expLanes will adapt to their absence, when possible: for example, without pdflatex, no reports will be generated.
