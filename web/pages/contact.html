There are various ways of getting involved in the project:

  • let us know if you enjoy expLanes: post us a message on explanes@googlegroups.com .
  • ask for assistance: use the mailing list explanes@googlegroups.com, please browse the forum for similar issues before submiting your question.
  • Report bug: use the github issue tracker with a complete description of the error together with an detailed description of your environment (version of matlab, operating system, etc.).
  • Submit a pull request: use gitHub pull tracker.
  • You want to contribute to expLanes: Welcome !! Please post us a message on explanes@googlegroups.com.
